Tools¶

Matlab - Octave¶

The following scripts used for the pre/post-treatment of AMITEX_FFTP input/output can be downloaded :

  • savefieldvtk.m to write .vtk file for AMITEX_FFTP input (material ID, zone ID or initial internal variables, see VTK files)
  • readfieldvtk.m to read .vtk file from AMITEX_FFTP output (stress, strain or internal variable fields, see Default Output)
  • savebin.m to write .bin file for AMITEX_FFTP input (per zone coefficients or initial internal variables, composite voxels etc…, see BIN files)
  • readbin.m to read .bin file (see BIN files)
